Bem-vindos, Casal Noronha!
Este é o sistema financeiro feito para vocês dois. Em 3 passos rápidos vamos deixar tudo configurado para começar.
📋
Primeiro: crie suas categorias
Categorias organizam seus gastos. Exemplos: Moradia, Alimentação, Transporte, Saúde, Lazer, Poupança.

Vá em Orçamento no menu e adicione as categorias com o valor que planejam gastar em cada uma.
🔁
Segundo: configure os recorrentes
Salário, aluguel, academia, plano de saúde — tudo que se repete todo mês. Configure em Recorrentes e o sistema lança automaticamente no início de cada mês.
＋
Terceiro: use o botão flutuante
O botão ＋ no canto inferior direito é o atalho mais rápido para registrar qualquer gasto ou receita. Você pode até falar e o sistema entende.

Tudo pronto. Boas finanças! 💛

❓ Ajuda & Tutorial
Como usar cada seção do sistema passo a passo
🚀 Por onde começar
1
Cadastre suas categorias
Vá em Orçamento → adicione categorias como Moradia, Alimentação, Transporte. As categorias padrão já foram criadas automaticamente para você.
2
Adicione seus cartões de crédito
Vá em Cartões de Crédito → adicione nome, limite e data de fechamento. Depois aparecerão como opção ao registrar despesas.
3
Configure suas despesas fixas
Vá em Recorrentes → adicione aluguel, contas de luz, academia, etc. Elas serão lançadas automaticamente todo mês no dia certo.
4
Registre gastos do dia a dia
Use o botão ＋ (canto inferior direito) para lançar receitas e despesas rapidamente. Ou use o 🎤 para falar/digitar em linguagem natural.
5
Acompanhe no Dashboard
O Dashboard mostra seu resumo do mês: receita, gastos, saldo e score financeiro. Use os botões de mês para navegar entre períodos.
📋 O que cada tela faz
💡 Dicas rápidas
🎤Clique no microfone e fale "gastei 85 reais no mercado, foi a Natália" — o app preenche tudo sozinho.
📅Os lançamentos recorrentes são criados automaticamente na virada do mês — você só confirma se foram pagos.
💳Ao registrar uma despesa no cartão, selecione o cartão no campo "Cartão" — o sistema calcula o ciclo de faturamento automaticamente.
🎯Em Metas, use o botão "Aportar" para depositar ou retirar valores. O progresso atualiza na hora.
🔒Configure um PIN de 4 dígitos em Menu → PIN de Acesso para proteger o sistema de olhares curiosos.
💾Os dados ficam salvos automaticamente na nuvem (Supabase) e sincronizam entre todos os dispositivos de vocês.
📊Use Menu → Exportar CSV para baixar todos os lançamentos em planilha (compatível com Excel).
👁Ative o Modo Leitura antes de mostrar o app para alguém — bloqueia edições e oculta botões de ação.
